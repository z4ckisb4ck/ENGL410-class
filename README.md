## Calculator

A simple command-line calculator that supports basic arithmetic operations.

### Features

- **Addition**: Add two numbers
- **Subtraction**: Subtract one number from another
- **Multiplication**: Multiply two numbers
- **Division**: Divide one number by another (with zero-division protection)
- **Power**: Raise a number to a power
- **Modulo**: Calculate the remainder of division

### Usage

Run the calculator from the command line:

```bash
python main.py
```

The calculator will display a menu where you can:
1. Select an operation (1-6)
2. Enter two numbers
3. View the result
4. Choose to perform another calculation or exit (0)

### Files

- `calculator.py` - Core calculator module with arithmetic functions
- `main.py` - Command-line interface for the calculator

### Requirements

- Python 3.6 or higher (no external dependencies required)
